# Объектная модель Sci-Summarizer ML сервиса

## Описание
Объектная модель ML-сервиса для суммаризации научных статей из PDF-файлов.

## Сущности системы

### Основные классы:
- **User** - пользователь системы с балансом
- **MLModel** - абстрактный класс ML модели
- **TextSummarizationModel** - конкретная реализация для суммаризации текста
- **PDFFile** - представление PDF файла
- **PredictionRequest** - запрос на обработку
- **Transaction** - финансовая транзакция

## Принципы ООП

### 1. Инкапсуляция
- Приватные поля (`_id`, `_balance`, `_password_hash`)
- Контролируемый доступ через свойства
- Валидация в сеттерах

### 2. Наследование
- `BaseEntity` ← `User`, `PDFFile`, `Transaction`, `PredictionRequest`
- `MLModel` ← `TextSummarizationModel`

### 3. Полиморфизм
- Абстрактные методы в MLModel
- Единый интерфейс `process()` для всех моделей
